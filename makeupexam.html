Program Description

You have been hired by the local news station that will help meteorologists simulate and predict monthly high Fahrenheit and Celsius temperatures here in Charlotte, NC.

According to usclimatedata.com, in 2023, Charlotte had the following average monthly high Fahrenheit temperatures:

January 51.0° F
February 55.0° F
March 63.0° F
April 72.0° F
May 79.0° F
June 86.0° F
July 89.0° F
August 87.0° F
September 81.0° F
October 72.0° F
November 62.0° F
December 53.0° F

Code a modular program that uses parallel arrays to store the monthly average high monthly Fahrenheit temperatures for Charlotte in 2023, randomly generated average high monthly Fahrenheit temperatures based on the 2023 average monthly temperatures and calculated monthly high Celsius temperatures based on the randomly generated average high monthly Fahrenheit temperatures.

The program must do the following:

  1. The program must have an array of type double named average2023FTemperatures that stores the monthly average high monthly Fahrenheit temperatures for Charlotte in 2023. Use an initialization list to store the values mentioned above.
  2. The program must have an array of type double named genAverageFTemperatures that stores the randomly generated monthly high Fahrenheit temperatures for the predictive analysis. Use a loop to populate the array by calling the generateRandFahrenheitTemp() function (function details below) and pass in each temperature from the average2023FTemperatures array.
  3. The program must have an array of type double named averageCTemperatures that stores the calculated monthly high Celsius temperatures for the predictive analysis. Use a loop to populate the array by calling the calcCelsiusTemp() function (function details below) and pass in each temperature from the genAverageFTemperatures array.
  4. The program must have a variable of type double named meanFahrenheitTemp that holds the calculated mean monthly Fahrenheit temperature using the array of generated monthly Fahrenheit temperatures, genAverageFTemperatures. This value must be calculated by calling the calcMean() function (function details below) and passing in the genAverageFTemperatures array.
  5. The program must have a variable of type double named meanCelsiusTemp that holds the calculated mean monthly Celsius temperature using the array of calculated monthly Celsius temperatures, averageCTemperatures. This value must be calculated by calling the calcMean() function (function details below) and passing in the averageCTemperatures array.
